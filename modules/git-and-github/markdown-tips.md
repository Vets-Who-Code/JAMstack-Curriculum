
# Markdown Tips

---

## Headers
- `# H1`
- `## H2`
- `### H3`
- `#### H4`
- `##### H5`
- `###### H6`

---

## Text Formatting
- `**bold**` or `__bold__`: **Bold Text**
- `*italic*` or `_italic_`: *Italic Text*
- `~~strikethrough~~`: ~~Strikethrough~~
- `> blockquote`: 
  > Blockquote
- `>> nested blockquote`: 
  >> Nested Blockquote

---

## Lists

### Unordered Lists
- `- item`
- `* item`
- `+ item`

### Ordered Lists
1. `1. First item`
2. `2. Second item`
3. `3. Third item`

### Task Lists
- `- [ ] Incomplete task`
- `- [x] Complete task`

---

## Links and Media

### Links
- `[Link text](URL)`
- `[Link text](URL "Title")`

### Images
- `![Alt text](Image URL)`
- `![Alt text](Image URL "Title")`

---

## Code

### Inline Code
- `` `Inline code` ``

### Fenced Code Blocks
- ` ```javascript `: JavaScript Code Block
- ` ```python `: Python Code Block
- ` ```bash `: Bash/Shell Code Block

### Indented Code Blocks
    Indent with four spaces.

## Horizontal Rules
- `---`
- `___`

## Tables
```
| Header 1 | Header 2 | Header 3 |
|----------|----------|----------|
| Cell 1   | Cell 2   | Cell 3   |

```

## Escaping Characters
- `\*`: To display a literal asterisk
- `\_`: To display a literal underscore

---

## HTML in Markdown
- `<span style="color:red">Text</span>`: Embed HTML for more control
